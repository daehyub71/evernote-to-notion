- **데이터 모델** (Python 코드)
- **테스트 계획**
- **예상 일정**
- **리스크 및 대응**

### TASKS.md (작업 바이블)
**목적**: 상세 Task 목록 (80+ Task)
**내용**:
- Phase별 Task 분류
- 각 Task별:
  - 상세 설명
  - 구현 코드 스니펫
  - 완료 기준
  - 체크박스

**예시 구조**:
```
Task 1.2: ENEX 파서 개발 (4-6시간)
  ├─ 1.2.1 EnexParser 클래스 구현
  │   └─ 코드 스니펫 (Python)
  ├─ 1.2.2 데이터 모델 정의
  │   └─ 코드 스니펫 (@dataclass)
  └─ 1.2.3 날짜/시간 파싱
      └─ 코드 스니펫 (datetime)
```

**특징**:
- 즉시 복사해서 사용 가능한 코드
- 단계별 가이드
- 예상 소요 시간 표시

**언제 읽나요**: 작업 중 항상 옆에 두고 참조
